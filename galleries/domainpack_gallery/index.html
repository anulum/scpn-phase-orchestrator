    Domainpack Gallery¶

    Each domainpack ships a binding_spec.yaml that maps a real-world problem onto SCPN Kuramoto oscillators. The notebooks below demonstrate baseline vs orchestrated simulations.

    Adding a New Domainpack¶

    1. Create domainpacks/<name>/binding_spec.yaml following the binding spec schema
    2. Add domainpacks/<name>/policy.yaml with supervisor rules
    3. Create a notebook in notebooks/ following the pattern above
