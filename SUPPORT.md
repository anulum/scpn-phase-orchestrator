## Before Opening an Issue

1. Search [existing issues](https://github.com/anulum/scpn-phase-orchestrator/issues)
2. Check the [documentation](https://anulum.github.io/scpn-phase-orchestrator/)
3. Try the latest version: `pip install --upgrade scpn-phase-orchestrator`

## Security Vulnerabilities

Do **not** open a public issue. Use [GitHub Security Advisories](https://github.com/anulum/scpn-phase-orchestrator/security/advisories) or email protoscience@anulum.li. See [SECURITY.md](SECURITY.md) for the full policy.
